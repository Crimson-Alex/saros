---Install Grunt---
Grunt depends on NodeJS and its package manager NPM.
For information about installing NodeJS see nodejs.org.
For more information about Grunt visit gruntjs.org.

1.  Run "npm install -g grunt-cli" to globally install the Grunt CLI.
2.  Navigate to "de.fu_berlin.inf.dpp.ui/resources/html/" and run "npm install"
    to install all node modules necessary for the specified Grunt tasks
    (dependencies are managed in package.json, tasks are specified in
    Gruntfile.js).

---Use Grunt---
Navigate to "de.fu_berlin.inf.dpp.ui/resources/html/" and run "grunt" to start
the tasks specified in Gruntfile.js.
