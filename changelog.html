DPP Changelog

This changelog includes some of the major feature additions/changes in every version. For more detailed information on refactorings, bug fixes, tests and minor functionality see the SVN history.

0.6.2

  • Project synchronization is done trough file transfers, which allows to share larger projects
  • Enable 'Share Project'-action also on Java projects
  • Show synchronization progress bar when joining a lesson
  • Made 'Create Account'-wizard more robust and show progress bar
  • Show correct connection icons in Roster View
  • Show status of connection in Roster View status line
  • Show presence in Roster View
  • Made automatically connect on startup optional (see preferences)
  • Handle malformatted Jabber IDs in 'Add Contact'-wizard
  • Added connection states CONNECTING, DISCONNECTING and ERROR
